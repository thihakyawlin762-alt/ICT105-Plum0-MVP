# Lab 09 User Consent Statement

This Smart Assignment Tracker prototype is developed for ICT105 coursework at Rangsit University. Users are invited to test the prototype and provide feedback to help improve the system. Participation is voluntary, and users may stop testing at any time. No unnecessary sensitive personal information should be provided during testing.

## Data Collected

- Student ID or anonymous tester ID
- User role (Student or Lecturer)
- Task completion result
- Feedback score (1–5)
- Interest level (1–5)
- Comments about usability
- Time taken to complete tasks

## Data Use

The collected information will be used only for:

- Evaluating the usability of the Smart Assignment Tracker prototype.
- Improving the prototype based on user feedback.
- Supporting analytics and validation for the ICT105 course project.
- Preparing project documentation and the final prototype.

Personal information will not be shared outside the project team or used for purposes unrelated to this coursework.
