# Customer Validation Summary

## Project Information

**Project Name:** Smart Assignment Tracker

**Validation Date:** 26–28 July 2026

**Prototype Version:** v1.0

---

## Test Users

The MVP prototype was evaluated by 20 participants consisting of students and lecturers. Students tested the core assignment management workflow, while lecturers tested the lecturer dashboard and assignment review functions.

| User Group | Number of Testers | Main Tasks |
|------------|------------------:|-----------|
| Students | 17 | Login, add assignments, search assignments, edit assignments, view dashboard |
| Lecturers | 3 | Lecturer dashboard, review assignments, view statistics |

---

## Tasks Tested

The following prototype features were validated during testing:

- Login
- Add Assignment
- Search Assignment
- View Assignment Details
- Edit Assignment
- View Dashboard
- View Completed Assignments
- Lecturer Dashboard
- Assignment Review
- Dashboard Statistics

---

## Evidence Collected

The team collected the following validation evidence:

- Task completion results
- Time required to complete each task
- Ease of use ratings
- Feature usefulness ratings
- User interest ratings
- User comments and suggestions
- Validation dataset
- Prototype screenshots

---

## Key Findings

The validation results indicate that most participants were able to complete the main assignment management tasks successfully without assistance.

Users found the Assignment Form, Search Function, Dashboard, and Assignment Details pages to be the most useful features.

Some participants experienced minor confusion when using the assignment search filters, editing assignments, and locating certain buttons. These issues are considered interface improvements rather than major functional problems.

Overall, the prototype successfully demonstrated the core workflow of the Smart Assignment Tracker and provided sufficient evidence to continue development with minor usability improvements.
