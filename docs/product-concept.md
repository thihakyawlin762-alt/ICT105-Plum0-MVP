# Product Concept

## 1. Project Title
Smart Assignment Tracker

## 2. Target User
Primary User: University students who need an organized way to manage assignments, deadlines, and academic tasks.

Secondary User: Lecturers/Administrators who oversee student assignment records, monitor progress, and manage assignment data when necessary.

## 3. Problem Statement
University students often struggle to keep track of multiple assignments, deadlines, and course requirements. Many rely on handwritten notes, messaging apps, or simple reminder tools that are scattered across different platforms. This makes it easy to forget deadlines, lose important assignment information, and manage academic work inefficiently.

## 4. Proposed IT Product
A web-based Smart Assignment Tracker that allows students to securely log in using their university account, manage assignments, organize deadlines, search and filter tasks, monitor completion status, and receive deadline reminders. The system also provides a lecturer/administrator interface for viewing assignment records and managing submitted data.

## 5. Core Value Proposition
The Smart Assignment Tracker provides students with one centralized platform to organize academic tasks, monitor deadlines, and track assignment progress. It improves productivity, reduces missed deadlines, and simplifies assignment management through an easy-to-use interface.

## 6. MVP Scope
The Minimum Viable Product (MVP) will include:

- University account login
- Dashboard overview
- Add new assignment
- Edit and delete assignments
- Assignment list with search and filter
- Assignment detail view
- Mark assignments as completed
- Deadline reminder notifications
- Lecturer/Administrator dashboard for assignment management
- Basic dashboard summary and assignment statistics

## 7. Out of Scope for Current Semester
- Google Calendar synchronization
- LMS (Learning Management System) integration
- Email or SMS notifications
- Group assignment collaboration
- AI-powered deadline prediction or smart scheduling
- Mobile application deployment
- Advanced authentication and security features
