# Initial IT Venture Idea Log

## Idea 1: Smart Assignment Tracker

### Problem Area
University students often miss assignment deadlines because information is scattered across LMS platforms, chat groups, emails, and classroom announcements.

### Target Users
University students.

### Current Alternative
Students manually check LMS systems, messaging applications, and class announcements.

### Proposed IT Solution
A web-based dashboard that collects assignments, deadlines, reminders, and notifications in one place.

### Possible Technology
HTML, CSS, JavaScript, database integration, notification system.

### Why This is Suitable
The system can be developed as a web prototype within one semester and validated using surveys from university students.

## Idea 2: Smart Campus Lost-and-Found System

### Problem Area
Students frequently lose personal belongings on campus and do not know where to report or search for them.

### Target Users
University students, campus security staff, and administrative staff.

### Current Alternative
Students ask friends, contact security personnel, or post messages in social media and chat groups.

### Proposed IT Solution
A web-based platform that allows users to report lost items, upload photos, search for found items, and contact responsible offices.

### Possible Technology
HTML, CSS, JavaScript, image upload feature, database or cloud storage.

### Why This is Suitable
The system is easy to prototype, addresses a real campus problem, and can be validated through interviews with students.

## Idea 3: SME Inventory Management System

### Problem Area
Small business owners often struggle to track inventory, sales records, and stock levels efficiently.

### Target Users
Small and medium-sized enterprise (SME) owners.

### Current Alternative
Paper records, spreadsheets, or manual counting.

### Proposed IT Solution
A simple web-based inventory management system that tracks stock levels, sales transactions, and low-stock alerts.

### Possible Technology
HTML, CSS, JavaScript, database integration, dashboard interface.

### Why This is Suitable
The system is practical, addresses a common business problem, and can be developed as a functional prototype during the semester.
