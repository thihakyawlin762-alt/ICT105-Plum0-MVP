# Lab 04 - MVP Feature List

## MVP Decision Rule

The MVP includes only the essential features needed to solve the main problem: helping students organize assignments and avoid missing deadlines.

---

|---|---|---|---|---|---:|---:|---:|---|
| F01 | Homepage / Landing Page | Introduces the system and its purpose | US-01 | Must | 1 | 4 | 5 | Yes |

## Must-Have Features

- Homepage / Landing Page
- Student Login
- Add Assignment
- Assignment Dashboard
- Edit / Delete Assignment

---

## Should-Have Features

- Search and Filter Assignments
- Assignment Detail Page
- Assignment Status (Pending / Completed)

---

## Could-Have / Future Features

- Reminder Notifications
- Dashboard Analytics
- Calendar Integration
- Email Notifications

---

## Not in MVP

The following features are intentionally excluded from the first prototype because they require more advanced development or were not strongly supported by the evidence collected during Lab 03:

- Automatic synchronization with university LMS
- Google Calendar integration
- Email or SMS reminder notifications
- AI-generated study recommendations
- Group assignment collaboration
- Multi-user sharing and teamwork features

These features may be considered in future versions after the core assignment management functions have been successfully implemented and validated.
