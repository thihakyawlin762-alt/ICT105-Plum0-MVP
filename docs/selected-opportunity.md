# Selected Semester Project Opportunity

## Selected Project Title

Smart Assignment Tracker

## Target User

University students who need to manage assignments, quizzes, projects, and academic deadlines from multiple courses.

## Problem Statement

Students often miss assignment deadlines because important information is scattered across different platforms such as LMS, chat groups, emails, and lecturer announcements. Managing deadlines manually can be confusing and inefficient.

## Current Alternative

Students currently use personal notes, calendars, screenshots, reminder applications, chat bookmarks, or repeatedly check LMS platforms and group chats to track assignment deadlines.

## Proposed IT Solution

The team will develop a Smart Assignment Tracker, a web-based application that allows students to record assignments, organize tasks by course, monitor upcoming deadlines, and view all academic tasks through a centralized dashboard.

## Why This Opportunity Was Selected

The Smart Assignment Tracker received the highest NUF score among the team's ideas. It addresses a common problem faced by university students, provides clear value, can be easily validated through student interviews and surveys, and is feasible to develop within one semester.

## Why This Is Feasible Without IoT or Advanced Cybersecurity

The project only requires software development technologies such as HTML, CSS, JavaScript, and Google Sheets or a simple database. It does not require sensors, hardware devices, IoT equipment, or advanced cybersecurity implementation.

## Possible MVP Type

Simple Web Application Prototype using HTML, CSS, JavaScript, and Google Sheets.

## Initial Validation Plan for Lab 03

### Who will the team ask?

University students from different faculties and year levels.

### What evidence will the team collect?

* Number of assignments students manage each week.
* Frequency of missed deadlines.
* Current methods used to track assignments.
* Problems with current tracking methods.
* Interest in using a centralized assignment tracking system.

### What question must be answered first?

Do university students experience enough difficulty managing assignment deadlines to justify the need for a dedicated assignment tracking system?
