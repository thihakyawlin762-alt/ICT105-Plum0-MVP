# Problem Notes | Lab-3

## Selected Case

Smart Assignment Tracker

## Target Respondents

| -------------------- | ------------------------------------------------------------- | -------------: | ------------: |
| First-year students  | Often adjusting to university workload and multiple platforms |              3 |             3 |

## Observed / Reported Problem

Many students struggle to keep track of assignment deadlines because information is spread across LMS platforms, chat groups, email announcements, and lecturer messages. Students often rely on memory, screenshots, or personal notes, which can cause missed or nearly missed deadlines.

| Workaround                   | Who Uses It? | Weakness / Pain                               | Evidence Source |
| ---------------------------- | ------------ | --------------------------------------------- | --------------- |

| Phone calendar               | Students     | Requires manual entry                         | Interviews      |
| Screenshots of announcements | Students     | Difficult to organize                         | Interviews      |
| Chat group reminders         | Students     | Messages get buried quickly                   | Interviews      |

## Notes for MVP Direction

The MVP should focus on a simple dashboard where students can view all assignment deadlines in one place. Priority features should include assignment entry, due date tracking, and reminder notifications. Advanced features such as AI recommendations or automatic LMS integration can be considered later after validating the core problem.
