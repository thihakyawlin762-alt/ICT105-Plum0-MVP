## Notes

This prototype was developed using **Sublime Text 3** with HTML5, CSS3, and JavaScript. Browser LocalStorage is used to simulate backend data storage for demonstration purposes. The complete working prototype is provided inside the **`SmartAssignmentTracker Lab 10`** folder within this repository. The project currently includes both the Student Module and the Lecturer Module, implementing the Minimum Viable Product (MVP) features required for Lab 10 – MVP Implementation Sprint 1.
