## Testing Notes

The Smart Assignment Tracker prototype is currently implemented as an HTML, CSS, and JavaScript web application.

Testers should begin by opening index.html and then complete the MVP experiment using the following workflow:

1. Open the homepage.
2. Log into the system.
3. Add a new assignment.
4. View the assignment list.
5. Search for an assignment.
6. Open the assignment details.
7. Edit the assignment information.
8. View the dashboard summary.
9. Complete the feedback form after finishing the experiment.

Notes

* The current MVP focuses primarily on the Student module.
* The Lecturer module is still under development and is not included in the MVP experiment.
* Assignment data is stored locally using LocalStorage, so no internet connection or backend server is required for testing.
* The prototype is intended for usability testing and demonstration purposes only.
