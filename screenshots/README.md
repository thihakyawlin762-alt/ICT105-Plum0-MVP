# Screenshots Lab 05 and Lab 06

## Purpose

This folder contains screenshots of the current development progress of the **Smart Assignment Tracker** prototype.

The screenshots are provided to document the implementation status during **Lab 05** and **Lab 06**, allowing reviewers to compare the prototype with the project documentation, wireframes, Functional Requirements (FR), and User Stories (US).

## Current Development Status

The screenshots represent the prototype at its current stage of development and do not necessarily reflect the final version of the system.

* The **Student Module** is approaching completion, with most core pages and features implemented.
* The **Lecturer/Administrator Module** is currently in the early stages of development and will continue to be expanded throughout the remaining project timeline.
